GIT & GITHUB:
"UN CONJUNTO PARA DESARROLLADORES"

Icono de GIT & GITHUB



La plataforma Github y el software Git pueden ser bastante útiles en el desarrollo WEB



Objetivo General:

Implementar el software Git y la plataforma Github en la creación de sitios web, aportando al trabajo colaborativo y la actualización del sitio web.



Objetivo Específico 1:

Identificar la utilidad de la plataforma Github, en cuanto a lo relacionado con el mantenimiento de repositorios de código.



Objetivo Específico 2:

Identificar la utilidad de la aplicación Git, como sofware de control de versiones para desarrolladores.



Objetivo Específico 3:

Conocer las herramientas que nos ofrecen los programas Git y Github, para control de versiones y creación de repositorios para los desarrolladores.





Es necesario conocer un poco de ambos software antes de entrar a estudiarlos
GIT GITHUB

Es un software para alojar proyectos utilizando el sistema de control de versiones. Aloja tu repositorio de código y te brinda herramientas muy útiles para el trabajo en equipo, dentro de un proyecto.

Facilita la colaboración con git. Es una plataforma que puede mantener repositorios de código en almacenamiento basado en la nube para que varios desarrolladores.
